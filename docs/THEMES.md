## 主题概述

pnana 提供了 6 种精心设计的主题，满足不同用户的视觉偏好和使用场景。

所有主题都经过优化，确保：
- **高对比度**：清晰的文本可读性
- **护眼设计**：长时间使用不疲劳
- **语法高亮**：清晰的代码结构
- **统一体验**：一致的 UI 元素样式

## 可用主题

### Monokai（默认）

经典的 Sublime Text 主题，深色背景，高对比度。

- **背景色**：深灰色
- **前景色**：浅色文本
- **特点**：
  - 高对比度，适合长时间编程
  - 经典的代码编辑器配色
  - 语法高亮清晰
- **适用场景**：日常编程、代码编辑

### Dracula

流行的深色主题，柔和的紫色调。

- **背景色**：深紫色
- **前景色**：浅色文本
- **特点**：
  - 柔和的紫色调，视觉舒适
  - 流行的开源主题
  - 护眼设计
- **适用场景**：长时间编程、夜间使用

### Solarized Dark

著名的精心设计的配色方案，深色版本。

- **背景色**：深蓝绿色
- **前景色**：浅色文本
- **特点**：
  - 科学设计的配色方案
  - 平衡的色彩对比
  - 适合各种编程语言
- **适用场景**：专业开发、长时间使用

### Solarized Light

Solarized 配色方案的浅色版本。

- **背景色**：浅蓝绿色
- **前景色**：深色文本
- **特点**：
  - 浅色背景，适合明亮环境
  - 科学设计的配色
  - 护眼设计
- **适用场景**：白天使用、明亮环境

### OneDark

Atom 编辑器的经典主题。

- **背景色**：深灰色
- **前景色**：浅色文本
- **特点**：
  - Atom 编辑器的经典配色
  - 柔和的色彩
  - 清晰的语法高亮
- **适用场景**：Atom 用户迁移、日常编程

### Nord

北欧风格的冷色调主题。

- **背景色**：冷色调深色
- **前景色**：浅色文本
- **特点**：
  - 北欧风格的冷色调
  - 简洁优雅
  - 护眼设计
- **适用场景**：喜欢冷色调的用户、长时间使用

## 切换主题

### 方法一：使用快捷键

1. 按 `Ctrl+T` 打开主题菜单
2. 使用方向键或 `k`/`j` 键导航
3. 按 `Enter` 应用选中的主题
4. 按 `Esc` 取消

### 方法二：通过配置文件

编辑 `~/.config/pnana/config.json`：

```json
{
  "theme": "dracula"
}
```

可用的主题值：
- `monokai`
- `dracula`
- `solarized_dark`
- `solarized_light`
- `onedark`
- `nord`

### 方法三：命令行参数

```bash
pnana --theme dracula file.txt
```

## 主题选择建议

### 根据使用场景

- **长时间编程**：Dracula, Solarized Dark, Nord
- **明亮环境**：Solarized Light
- **经典风格**：Monokai, OneDark

### 根据个人偏好

- **喜欢高对比度**：Monokai
- **喜欢柔和色调**：Dracula, OneDark
- **喜欢冷色调**：Nord
- **喜欢科学配色**：Solarized Dark/Light

### 根据文件类型

所有主题都支持语法高亮，但某些主题可能对特定语言显示效果更好：

- **通用编程**：所有主题
- **Web 开发**：Monokai, Dracula
- **系统编程**：Solarized, Nord

## 常见问题

### Q: 如何快速切换主题？

A: 使用 `Ctrl+T` 快捷键打开主题菜单，然后使用方向键选择。

### Q: 主题设置会保存吗？

A: 是的，主题选择会保存到配置文件中，下次启动时自动应用。

### Q: 可以同时使用多个主题吗？

A: 不可以，一次只能使用一个主题，但可以为不同文件类型设置不同主题（未来功能）。

### Q: 主题会影响性能吗？

A: 不会，主题只是颜色配置，对性能没有影响。
