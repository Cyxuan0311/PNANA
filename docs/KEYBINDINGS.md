## 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建文件 |
| `Ctrl+O` | 切换文件浏览器 |
| `Ctrl+S` | 保存文件 |
| `Alt+A` | 另存为 |
| `Ctrl+W` | 关闭标签页 |
| `Alt+F` | 创建文件夹 |
| `Alt+M` | 打开文件选择器 |
| `Ctrl+Q` | 退出编辑器 |
| `Alt+Tab` / `Ctrl+PageDown` | 下一个标签页 |
| `Alt+Shift+Tab` / `Ctrl+PageUp` | 上一个标签页 |

### 文件浏览器内

| 操作 | 功能 |
|------|------|
| `↑` `↓` | 浏览文件列表 |
| `Enter` | 打开文件/文件夹 |
| `Backspace` | 返回上级目录 |
| `Tab` | 切换类型筛选 |
| `F6` | 移动文件/文件夹 |
| `Ctrl+L` | 分屏打开选中文件（需先选中文件） |

## 编辑操作

### 撤销 / 重做

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` / `Ctrl+Shift+Z` | 重做 |

### 剪贴板与选择

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+X` | 剪切 |
| `Ctrl+P` | 复制 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+A` | 全选 |
| `Alt+D` | 选中当前单词 |
| `Ctrl+→` / `Ctrl+←` | 自由选择（按单词移动并选中） |
| `Shift+↑` `↓` `←` `→` | 选择文本 |
| `Alt+Shift+↑` `↓` `←` `→` | 扩展选区 |

### 行操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+D` | 复制行 |
| `Ctrl+Shift+K` | 删除行 |

> **说明**：`Ctrl+L` 在文件浏览器选中文件后为分屏打开；在代码区分屏存在时显示关闭分屏对话框。

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Backspace` | 删除光标前单词 |
| `Tab` | 缩进行 |
| `Shift+Tab` | 减少缩进 |
| `Ctrl+/` | 切换注释 |
| `Alt+↑` | 上移行 |
| `Alt+↓` | 下移行 |

### 搜索

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+F` | 搜索 |
| `Ctrl+H` | 替换 |
| `Ctrl+F3` | 查找下一个 |
| `Ctrl+Shift+F3` | 查找上一个 |

### LSP（需启用 LSP 支持）

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Space` | 触发代码补全 |
| `Ctrl+U` | 切换折叠 |
| `Ctrl+Shift+U` | 折叠全部 |
| `Ctrl+Alt+U` | 展开全部 |

## 视图与工具

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+T` | 主题菜单 |
| `Ctrl+B` | 符号导航（LSP 文档符号） |
| `Alt+E` | 错误诊断 |
| `F1` | 显示/隐藏帮助 |
| `Ctrl+Shift+L` | 切换行号 |

### 图片预览与终端

- 在代码区**选中图片文件名**可自动预览
- 在命令面板中选择「terminal」可打开集成终端
- 终端内 `+` / `-` 调整高度

|--------|------|
| `Alt+P` | 打开插件管理器 |

## 模式与对话框

### 主题菜单

- `↑` `↓` 或 `k` `j`：选择主题  
- `Enter`：应用  
- `Esc`：取消  

### 搜索 / 替换

- `Enter`：执行  
- `Esc`：取消  

### 跳转到行

- 输入行号后 `Enter` 跳转  

### 另存为 / 创建文件夹

- `Enter`：确认  
- `Esc`：取消
