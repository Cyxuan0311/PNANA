## 版本历史

### v0.0.4（当前版本）

**发布日期**：2025-01-07

**已完成功能**：

- [x] 基础编辑
  - 文本输入、光标移动、选择
  - 撤销/重做、基本编辑操作
- [x] 多文件与分屏
  - 标签页、多文档切换
  - 水平/垂直分屏（Ctrl+L）
  - 分屏区域间焦点切换
- [x] 搜索
  - 文本搜索、正则表达式
  - 搜索结果高亮
- [x] 语法高亮
  - 多语言（tree-sitter + 内置）
  - 文件类型识别
- [x] 主题与配置
  - 多种内置主题
  - JSON 配置、命令行参数
- [x] LSP 支持
  - 代码补全、实时诊断
  - 跳转到定义、查找引用
  - 符号搜索、代码格式化
- [x] Lua 插件系统
  - 插件 API、事件、命令、键位映射
  - 插件管理（启用/禁用）
  - 主题扩展、状态栏美化
- [x] 终端集成
  - 内置终端
  - 命令执行、输出显示
- [x] 文件浏览器
  - 目录树、文件导航
- [x] Todo 面板
  - 待办管理、到期提醒
- [x] 图片预览
  - FFmpeg 转 ASCII 艺术预览
- [x] SSH 模块
  - Go 实现的 SSH 连接（待完善测试）

## 贡献指南

欢迎参与 pnana 开发：

1. **查看 Issues**：了解待解决问题
2. **选择任务**：挑选感兴趣的功能或 bug
3. **提交 PR**：按贡献指南提交 Pull Request
4. **参与讨论**：在 Issues 中参与功能讨论

### 参与步骤

1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/AmazingFeature`
3. 提交更改：`git commit -m 'Add some AmazingFeature'`
4. 推送分支：`git push origin feature/AmazingFeature`
5. 开启 Pull Request
